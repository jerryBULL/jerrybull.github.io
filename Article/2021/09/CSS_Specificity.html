前言

權重的意思就是CSS的優先權。

當兩個選擇器同時作用在一個元素，權重高的優先生效。

注意事項

  • 由權重低依序向上介紹。
  • 權重大小最左邊的權重最高，最右邊的權重最低。

    ex：0-0-0-1 < 0-0-1-0

  • 如果權重相同時，越後面寫的的會覆蓋掉前面的。

各個權重介紹

*

0-0-0-0

全站預設值。

element

0-0-0-1

element種類很多，例如：div、ul、li....等，可查看 HTML元素。

attribute（屬性選擇器）

0-0-1-0。

[type=email]、[type:checkbox]、[attr]...等

psuedo-class（偽類）

0-0-1-0。

:link、:hover、:focus、:nth-child()...等。

class

0-0-1-0

id

0-1-0-0

inline style

1-0-0-0

直接寫在Html行內的style。

!important

1-0-0-0-0

最強大的存在，只有自己(!important)能夠超越自己。

使用時需要謹慎使用，不然當樣式多了會很難維護。
