前言

權重的意思就是CSS的優先權。

相同權重的情況下後寫的CSS可以覆蓋先寫的CSS。

當兩個選擇器同時作用在一個元素，權重高的優先生效。

各個權重

*

0-0-0-0

全站預設值。

Element

0-0-0-0

Element種類很多，例如：div,ul,li,ol....等，可查看HTML元素。

attribute（屬性選擇器）

0-0-0-0

全站預設值。

psuedo-class(偽類)

0-0-0-0

全站預設值。

class

0-0-0-0

全站預設值。

ID

0-0-0-0

全站預設值。

inline style

0-0-0-0

全站預設值。

!important

0-0-0-0

全站預設值。
